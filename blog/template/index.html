Creating New Posts

To create a new blog post, follow these steps:

  1. Copy the entire blog/posts/template directory
  2. Rename it to your desired slug (e.g., my-first-post)
  3. Edit the index.html file with your content
  4. Run ./link.sh to update the blog index

Styling Consistency

The blog inherits the same fonts, colors, and spacing as the main Noosphere Analytics site. This ensures a consistent user experience across all sections. The responsive design adapts to mobile devices automatically.

Best practices for blog content:

  • Use semantic HTML elements
  • Keep paragraphs short and readable
  • Include relevant images when appropriate
  • Add meta descriptions for search engines
  • Use proper heading hierarchy

Next Steps

Now that you've seen the template, you can start creating your own content. The beauty of this system is its simplicity: no databases, no CMS, just HTML files and a shell script to keep everything organized.
